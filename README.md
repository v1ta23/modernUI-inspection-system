# 简介

## 设备点检记录系统

> C# WinForms 桌面应用 | SQL Server | ADO.NET

面向小型制造企业的设备点检管理工具，支持登录鉴权与实时系统资源监控。

**技术栈：** .NET / C# · WinForms · SQLite · ADO.NET
